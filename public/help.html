How to use our site

The Dashboard

In the dashboard, you can view your three most recently used functions. To see more information, edit, use, or share the functions, click on the button next to the formula. To see all of your functions, click on View All. This will take you to a landing, where you can search your functions, sort by tag, and more.

Creating Functions

From either dashboard, select the + icon in the bottom right corner. This will allow you to edit a new blank function. Notice that your function equation will look like \( \). These are the typical LaTeX in-line delimeters, so please do not remove them, or otherwise your function will not parse! Simply place your LaTeX code between the delimeters.

Edit/Use Function Page

From this page, you can view, use, and edit your function. You will reach a page that looks like this:


As you can see, you have the choice of entering a function here. Be sure to separate any multipied variables with a multiplication operator, as the parser will not be able to tell if "mc" is one variable or two! Also, be sure to click submit whenever changes to the equation are made(this is to avoid accidental changes).

Viewing Functions

When viewing all of your functions, they will automatically be sorted in order of creation. To view tagged functions, select the desired tag from the side menu.

Explore

This page allows you to search functions. This search will search function names, tags, and descriptions. To use a function created by another user, click to add a copy to your account first.

Profile Page

From this page, you can view your profile details, including username and email.
